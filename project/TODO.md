# 모여봐요 동물의 숲 아이템 카탈로그 구현 티켓

## 프로젝트 개요
순수 HTML, CSS, JavaScript를 사용하여 '모여봐요 동물의 숲' 게임의 아이템(가구, 옷, 신발, 가방, 도구 등)을 카탈로그 형태로 보여주는 웹사이트 구현

## 📋 기술 스택
- HTML5
- CSS3 (Flexbox/Grid, Animations, Media Queries)
- Vanilla JavaScript (ES6+)

## 🎨 주요 기능
1. 🏠 메인 홈 화면 (아이템 카테고리 버튼)
2. 📦 카테고리별 아이템 카탈로그
3. 🍃 클릭 시 나뭇잎 이모지 효과
4. 🌿 동물의 숲 테마 배경 이미지
5. 🔍 아이템 검색/정렬 (선택사항)
6. ❤️ 즐겨찾기 기능 (선택사항)

## 🎮 아이템 카테고리
- **가구 (Furniture)**: 침대, 의자, 테이블, 장식품 등
- **옷 (Clothing)**: 셔츠, 원피스, 모자 등
- **신발 (Shoes)**: 운동화, 부츠, 슬리퍼 등
- **가방 (Bags)**: 백팩, 핸드백, 파우치 등
- **도구 (Tools)**: 낚싯대, 삽, 도끼, 물뿌리개 등
- **기타 (Other)**: 벽지, 바닥재, 음악 등

## 🎨 디자인 컨셉
- **색상**: 파스텔 톤 (연두색, 민트색, 베이지색, 흰색)
- **폰트**: 둥글고 귀여운 느낌
- **레이아웃**: 카드 기반 그리드
- **애니메이션**: 부드럽고 자연스러운 전환

## 📝 참고사항
- 실제 게임 데이터는 사용하지 않고 샘플 데이터 사용
- 이모지를 아이템 아이콘으로 활용
- 모바일 우선 반응형 디자인
- SPA 방식으로 페이지 전환
