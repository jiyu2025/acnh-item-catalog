# 🦉 부엉이의 기록 - 동물의 숲 아이템 카탈로그 기획서

## 1. 프로젝트 개요

### 1.1 기본 정보
- **프로젝트명**: 부엉이의 기록 (Owl's Records)
- **부제**: 동물의 숲 박물관 관장 부엉의 비밀스러운 아이템 백과사전
- **컨셉**: 박물관 관장 부엉의 비밀 백과사전, 차가운 데이터 나열이 아닌 '살아있는 아카이브', 게임 속 박물관의 따뜻한 감성을 웹 UI로 재해석
- **목적**: 동물의 숲 아이템을 인터랙티브한 카탈로그로 탐색하고 관리
- **타겟**: 동물의 숲 팬, 아이템 수집가, 인테리어 관심 유저
- **구현 방식**: SPA (Single Page Application)


### 1.1.2 캐릭터 프로필
- **부엉**: * 박물관 관장 / 기록의 주체
검색 가이드 및 데이터 부재 시(Error) 사과하며 당황하는 코믹한 역할 담당

-**부옥**: 부엉의 여동생 / 감성적 조력자
위시리스트 권장 및 희망적인 멘트로 사용자의 수집욕 자극

### 1.2 핵심 가치
- 🎨 **감성적 경험**: 게임 세계관을 반영한 부엉이 캐릭터와 인터랙션
- 💚 **직관적 UI**: 카테고리 기반 탐색, 강력한 검색, 간편한 위시리스트
- 📱 **반응형 디자인**: 모바일부터 데스크톱까지 완벽 대응
- 💾 **데이터 저장**: 사용자 취향을 기억하는 로컬 스토리지 활용

**레이아웃**
- **헤더**

  - 로고: 우측 상단 (180x180px)

  - 카드 디자인: 크림색 배경, 둥근 모서리, 그림자 효과
  - 6개 카테고리:
    - 가구 🛋️ (서브카테고리 존재)
    - 옷 👕
    - 신발 👟
    - 가방 🎒
    - 도구 🔨
    - 기타 🎵
  
- **고정 UI 요소**
  - 위시리스트 버튼: 우측 고정 (top: 220px, right: 20px)
  - 찜 개수 뱃지: 빨간 원형, 숫자 표시
  - 부엉이 캐릭터: 좌우 하단 고정 (메인 화면에서만 표시)

    - 우측 boo: 200px 크기

**인터랙션**

- 검색창 포커스/호버:
  - 캐릭터 위로 20px 튀어오름

**아이템 카드**

  - 이미지: 상단, 정사각형
  - 이름: 1.1rem, Jua 폰트

- 클릭: 색상 변형이 있으면 모달 오픈

- 소파 (5개 아이템)
- 의자 (5개 아이템)
- 테이블 (5개 아이템)
- 램프 (5개 아이템)

**레이아웃**
- 서브카테고리 버튼: 상단, 가로 스크롤
- 아이템 그리드: Category Page와 동일
- 뒤로 가기: 홈으로 복귀

#### 🎨 Variant Modal
**트리거**
- 큐트 시리즈 아이템 클릭 시 (침대, 소파, 의자, 테이블, 램프)

**구성**
- 배경: 반투명 검정 오버레이 (backdrop-filter: blur 10px)
- 모달: 중앙 정렬, 흰색, 둥근 모서리
- 타이틀: "색상을 선택하세요" + 닫기 버튼 (✕)
- 색상 그리드: 2열, 4가지 색상 (스카이블루, 화이트, 핑크, 옐로우)
- 슬라이드인 애니메이션: 하단에서 위로 (0.3초)

#### ❤️ Wishlist Modal
**트리거**
- 우측 고정 위시리스트 버튼 클릭

**구성**
- 전체 화면 오버레이
- 헤더: "나의 위시리스트 ❤️" + 닫기 버튼
- 아이템 그리드: 2열 → 3열(768px+) → 4열(1024px+)
- 빈 상태: 부엉이 이미지 + 안내 메시지

**기능**
- 카테고리 뱃지 표시
- 하트 클릭으로 즉시 제거
- localStorage 동기화

#### 🔍 Search Results
**표시 조건**
- 검색어 입력 후 Enter 키

**구성**
- 검색어 헤더: "검색 결과: [검색어]"
- 카테고리 뱃지: 각 아이템에 표시
- 전체 카테고리 통합 검색

**결과 없음**

- 6개 노란 느낌표: 이미지 주변 원형 배치

- 애니메이션: 느낌표 팝업 (1.5초, 0.1초 간격 딜레이)

## 6. 사용자 시나리오

### 6.1 첫 방문 사용자
1. 페이지 로드 → 부엉이 캐릭터와 말풍선 확인
2. 6개 카테고리 카드 탐색
3. "가구 🛋️" 클릭 → 5개 서브카테고리 확인
4. "침대" 선택 → 5개 아이템 그리드 표시
5. "드림 큐트 침대" 클릭 → 색상 모달 오픈
6. 하트 버튼 클릭 → boo 반응 + 위시리스트 추가
7. 우측 위시리스트 버튼 클릭 → 찜 목록 확인

### 6.2 재방문 사용자
1. 페이지 로드 → 위시리스트 개수 표시 (localStorage)
2. 검색창 클릭 → 최근 검색어 태그 표시
3. "소파" 태그 클릭 → 바로 검색 실행
4. 검색 결과에서 새 아이템 발견
5. 하트 버튼으로 추가 수집
6. 위시리스트 모달에서 전체 컬렉션 확인

### 6.3 탐색 사용자
1. 검색창에 "램프" 입력 후 Enter
2. 전체 카테고리에서 "램프" 포함 아이템 표시
3. 결과 없는 검색어 입력 → boom.png + 느낌표 애니메이션
4. 검색창 포커스 → 부엉이 반응 확인
5. 페이지 클릭 → 나뭇잎 효과 즐기기

## 7. 기술 스택
