## 📋 기술 스택
- HTML5
- CSS3 (Flexbox/Grid, Animations, Media Queries)
- Vanilla JavaScript (ES6+)

## 🎨 주요 기능
1. ✨ 45개의 로또 공이 자유롭게 떠다니는 히어로 섹션
2. 🎲 로또 번호 자동 생성 (1~45 중 6개, 중복 없음)
3. 📊 생성된 번호 히스토리 관리
4. 💾 로컬 스토리지를 통한 데이터 영속성 (선택사항)

## 📝 참고사항
- 모든 번호는 1부터 45까지의 범위
- 생성된 6개의 번호는 중복되지 않음
- 번호는 오름차순으로 정렬하여 표시
- 로또 공의 색상은 번호 범위에 따라 구분 가능 (예: 1-10: 노랑, 11-20: 파랑, 등)
